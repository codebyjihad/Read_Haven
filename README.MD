# ReadHaven - Book Store Application

ReadHaven is a modern online book store where users can browse, search, and manage books. It is built with a **Node.js and Express backend** and a **React + Vite frontend**. The project uses **MongoDB** for data storage and **TailwindCSS** for styling.

---

## Features

- User-friendly interface to browse and search books  
- Add, update, and delete books (admin functionality)  
- Responsive design with TailwindCSS  
- Form handling with React Hook Form  
- Notifications using React Toastify  
- Routing with React Router  
- Backend API built with Express and MongoDB  

---

## Tech Stack

### Frontend
- React + TypeScript  
- Vite (build tool)  
- TailwindCSS (styling)  
- React Router (routing)  
- React Hook Form (forms)  
- React Toastify (notifications)  

### Backend
- Node.js + Express  
- MongoDB (database)  
- dotenv (environment variables)  
- cors (cross-origin requests)
